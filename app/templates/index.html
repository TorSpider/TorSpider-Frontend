Open-Source and Community Supported.

Open Source

The TorSpider spiders, the backend API, and the frontend web services are all open-source.

Community Supported

TorSpider is supported by the community. We are an open initiative where members can contribute their computing resources as spiders or donations to help fund infrastructure.

Flexible Architecture

TorSpider was built to grow as the community matures. With a back-end API used by both the spiders and the front-end website, we can upgrade, scale, and replace parts as needed.
